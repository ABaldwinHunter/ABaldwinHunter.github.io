Splitting strings into pieces in an array

split("")

(Note that you can add a character to the .split argument and it will cut the string at the points where it encounters the character. In the example to the right, we use a space)

 "Hello darling, your blue wig looks awesome today!".split(" ") => ["Hello", "darling,", "your", "blue", "wig", "looks", "awesome", "today!"]
